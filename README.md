# Informed Trading in Prediction Markets

A two-stage ML pipeline to detect anomalous pre-event trading on Polymarket and Kalshi.

## Structure
- `notebooks/` — Analysis notebooks (01 data collection → 04 results)
- `data/` — Local only, not tracked by git

pip install pandas pyarrow scikit-learn xgboost shap matplotlib seaborn requests tqdm
